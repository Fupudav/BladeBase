Sauvegarde locale

Les coches, statuts, doublons, notes, prix d'achat, valeurs estimées, dates d'achat, photos personnelles et priorités sont sauvegardés automatiquement dans ce navigateur. Les mises à jour de l'application ne suppriment pas ces données.

Mode application

Sur Android/Chrome : ouvre le site GitHub Pages, menu ⋮, puis Ajouter à l'écran d'accueil. BladeBase fonctionne hors ligne après la première ouverture et se met à jour automatiquement quand une nouvelle version est publiée. Pour éviter les blocages GitHub Pages, l'application ne récupère plus de photos sur Internet : ajoute les images officielles dans le dossier images/.
